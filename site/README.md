# Локальная копия сайта Siberian motorbears

Чистый HTML + CSS, минимум JavaScript (карусели Swiper, полноэкранный просмотр фото).

## Как открыть

- Открыть в браузере файл `index.html` (двойной клик или «Open with»).
- Или запустить локальный сервер из папки `site`, например: `python3 -m http.server 8000` и открыть http://localhost:8000

## Структура

- `index.html` — главная
- `order/` — производство
- `baribal/`, `panda/` — модели (покупка)
- `rent/` — аренда
- `baron/`, `mia/` — аренда конкретных моделей
- `contact/` — контакты
- `legal/` — реквизиты
- `examples/` — примеры работ
- `css/main.css` — стили
- `js/carousel.js` — карусели: листание, перемешивание фото при загрузке, полноэкран по клику
- `assets/img/` — изображения

## При выкладке новой версии сайта

- Чтобы сохранить подключение **Яндекс.Метрики**, **Яндекс.Вебмастера** и **Google Search Console**, в новую вёрстку нужно перенести те же коды счётчика и проверочные метки/файлы. Подробно: **docs/DEPLOY_METRICS_AND_WEBMASTERS.md**.

## При добавлении новых страниц

- **Всегда проверять и при необходимости обновлять `sitemap.xml`** — добавить новый URL с `lastmod`, `changefreq` и `priority`, иначе новая страница может хуже индексироваться.
- По желанию проверить остальное: canonical, meta title/description/keywords, Open Graph и Twitter Card, favicon в `<head>`, хлебные крошки (визуально и JSON-LD BreadcrumbList).

## Карусели

- На каждой странице с галереей фото при открытии порядок слайдов случайно перемешивается.
- Листание: стрелки, свайп, клавиши ←/→.
- Клик по фото открывает полноэкранный просмотр с переключением и закрытием по Escape или клику по фону.
